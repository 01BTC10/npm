npm-help(1) -- Get help about npm commands
==========================================

## Usage

    npm help <section>

Where <section> is one of:
  
  * activate
  * adduser
  * config
  * deactivate
  * folders
  * help (this page)
  * install
  * json
  * link (or ln)
  * list (or ls)
  * publish
  * registry
  * tag
  * uninstall (or rm)
  * build
  * npm
  * scripts
  * json
